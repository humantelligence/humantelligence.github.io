1. Major Features Upgrade

1.1 Admin and Organizational User Experience Upgrade


New Group Filter

It’s been included a type-ahead completer for group names on Admin -> Users and Organization -> Culture Analytics



This filter is available at:

  • Admin -> Users
  • Admin -> Groups
  • Organization -> Culture Analytics
  • Organization -> Cultere Analytics -> Compare Segments



New Direct Link Copy to Clipboard Tool

It’s been included a new action on the Admin -> Users list. That action allows to copy the user’s direct_link (invitation link) with a single click to be used on emails or notifications.


Import/Export User Template Update

A small change on the field names in the import and export template have been introduced in favor of the Talent Fit Terminology.

The field position_title have been renamed to job_title. The position of this field on the template is the same. Please note that the new field name is required, if the correct name is not supplied, the system will not work as expected.
